# BlackRoad Particle System

GPU-friendly particle system simulation with physics fields and ASCII rendering.

## Features

- **Particles**: Full physics with position, velocity, acceleration, rotation, mass, RGBA color
- **Emitters**: Burst, continuous, and explosion emission types
- **Force Fields**: Gravity, wind, vortex, attractor, repulsor, turbulence
- **ASCII Renderer**: Real-time ASCII art frame output
- **Presets**: Fire, explosion, fountain systems

## Usage

```bash
python particle_system.py
```
